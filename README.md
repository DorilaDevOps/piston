# Radial Piston Dots

Visualización interactiva de puntos oscilando sobre diámetros de un círculo. Cada punto se mueve de extremo a extremo de su propio diámetro a velocidad constante, con arranque escalonado tipo ola.

## Demo

Abrí `index.html` en cualquier navegador moderno.

## Características

- N puntos distribuidos en N diámetros únicos (ángulo = 180°/N)
- Oscilación constante de extremo a extremo
- Arranque escalonado — los puntos salen uno tras otro
- Velocidad ajustable (9 niveles: 0.3 – 60)
- Agregar / quitar puntos (mín. 2)
- Tema oscuro / claro
- Responsive (mobile-first)
- ARIA labels y soporte SEO

## Controles

| Botón | Acción |
|-------|--------|
| `+` / `−` | Agregar / quitar punto |
| `⟵` / `⟶` | Disminuir / aumentar velocidad |
| `🌙` / `☀️` | Alternar tema |
